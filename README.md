# Site Singularidades - Links Úteis

Site com links e recursos educacionais do Instituto Singularidades, otimizado para GitHub Pages.

## Estrutura

- `index.html` - Página principal com todos os links
- `style.css` - Estilos modernos e responsivos
- `Arquivos/` - Pasta com documentos PDF

## Como publicar no GitHub Pages

1. Faça commit de todos os arquivos para o repositório
2. Vá em Settings > Pages no seu repositório GitHub
3. Selecione a branch `main` (ou `master`) como source
4. O site estará disponível em `https://seu-usuario.github.io/nome-do-repo`

## Recursos

O site inclui links para:
- Matriz de Referência do ENADE 2025
- BNCC e LDB (com opções de visualização e download)
- BNCC Somente
- BNCC Computação
- Perfil do Egresso Singularidades
- Site de Estágio
- Ementário 2025
